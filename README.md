## Run Locally

Clone the project

```bash
  git clone https://github.com/developerwaleed/VisionExpress.com.git
```

Go to the project directory

```bash
  cd VisionExpress.com
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx eslint . --fix
  npm run start
```

Run the tests

```bash
  `npm run test`
```
